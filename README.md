# Lightloop

Generative UI Chat App built with CopilotKit, LangGraph, and TanStack Start.

## Tech Stack

- **Frontend**: TanStack Start + Tailwind CSS
- **Backend**: TanStack AI SDK + CopilotKit + LangGraph  
- **Database**: Bun SQLite (bun:sqlite)
- **Auth**: Clerk
- **Runtime**: Bun
